# Audio Spectrum Video Creator

Aplikasi web berbasis browser untuk membuat video visualisasi spektrum musik secara *real-time*. Anda dapat mengunggah musik, gambar latar, dan logo untuk menghasilkan video profesional langsung dari browser Anda.

## 🚀 Fitur Utama
- **Multi-Playlist Audio:** Upload banyak lagu sekaligus dan putar secara berurutan.
- **Custom Visualizer:** Tersedia 4 gaya spektrum (Bar Klasik, Bar Tengah, Lingkaran, dan Waveform).
- **Branding:** Tambahkan logo dengan pengaturan posisi dan ukuran yang fleksibel.
- **Video Export:** Rekam visualisasi Anda secara langsung dan simpan dalam format video (.webm).
- **No Server Needed:** Semua pemrosesan dilakukan di sisi klien (*client-side*).

## 🛠 Cara Penggunaan
1. Buka [Live Demo](#) **(Ganti tanda # dengan link GitHub Pages Anda)**.
2. Upload gambar latar (background) dan file audio (playlist).
3. Atur posisi logo dan gaya spektrum melalui panel kontrol.
4. Klik **"▶ Mulai Rekam Video"**.
5. Setelah selesai, klik **"⏹ Berhenti & Simpan Video"** untuk mengunduh hasil rekaman Anda.

## 💻 Teknologi yang Digunakan
- **HTML5 Canvas API:** Untuk rendering visual.
- **Web Audio API:** Untuk analisis frekuensi suara.
- **MediaRecorder API:** Untuk merekam stream ke format video.
